We Rate Dogs Twitter Analysis

Project Details

This data wrangling project is the second of my Udacity Data Analysis program, which is aimed to showcase my ability to gathering, assessing, cleaning, and analyzing a dataset.

WeRateDogs is a Twitter account that rates people’s dogs with a humorous comment about the dog. The ratings usually have a denominator of 10. The twitter account has garnered over 9 million followers and has been active since November 2015. With over 5000+ tweets from the Twitter account, the analysis would help us answer popular questions from what makes a dog tweet get high likes to what dog breed gets the most engagements.

Project Goal

The primary goal of this project is to effectively wrangle the @weratedogs twitter data in relation to dog ratings. Not all data points in the tweet archive were extracted correctly though, which is why this needs to be treated rightly in the wrangling steps. So what is wrangling? Data wrangling refers to the process of cleaning, restructuring and enriching the raw data available into a more usable format. I believe further analysis can be carried out in the future after this wrangling with a much broader exploration. The act_report of this project communicates the insights and visualizations produced from the wrangled data.

Tools Used

  • Jupyter notebook
  • Python libraries (Pandas, Numpy, Seaborn, Matplotlib)
